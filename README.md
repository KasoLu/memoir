# 古法 Memoir

[English](README_EN.md)

一款 SillyTavern 扩展，为长聊天提供**可控的记忆管理**——手动分段总结、确认注入、融合压缩。让 AI 在续写时始终记住之前的剧情，而不是随着聊天变长逐渐遗忘。

> "古法"不是只能手动，而是你的记忆你做主——不搞黑盒，不偷偷删聊天。

---

## 工作原理

```
选定楼层区间
       ↓
LLM 按照总结提示词生成剧情档案
       ↓
你预览、编辑后确认采用
       ↓
已确认的总结自动注入到对话上下文
       ↓
下次 AI 回复时就能"记住"之前的剧情
```

---

## 功能亮点

- **手动分段总结** — 你来选区间，你来确认，不自动乱删聊天记录
- **总结预设 + 风格/同人/成熟内容兼容模板** — 内置剧情档案预设，可叠加风格模板（轻松自然、细节存档、剧情骨架、续演友好）、同人二创模板和成熟内容归档兼容补丁，模板正文完全可查看和编辑，也可以替换成你自己的惯用总结模板
- **空回恢复** — 总结或融合压缩遇到空响应时会自动重试；仍无可用结果时，会保留可编辑的兜底正文，避免总结流程中断
- **融合压缩** — 多次总结累积太长？一键融合压缩，去重合并；如果供应商持续空回，会保留原始已选总结作为手动修订文本
- **相对注入策略** — 选择注入层（最前层 / 角色设定层 / 聊天注入层）、层内深度、角色，支持自定义包裹标签；聊天注入层按相对顺序参与酒馆排序，不承诺绝对贴着最新消息
- **共享/独立 API** — 可以用酒馆已配置的 API，也可以配独立的总结专用 API
- **Fetch Models** — 一键获取独立 API 的可用模型列表
- **全屏编辑** — 长文本提示词和总结正文支持全屏编辑，总结正文全屏还带搜索
- **主题切换** — 内置 7 套面板主题色，悬浮球颜色跟随；面板颜色独立于 SillyTavern 原生主题，深色/浅色面板都保持可读
- **防误关面板** — 面板外点击和悬浮球二次点击不会关闭面板，避免总结或编辑中途丢内容
- **移动端适配** — 面板、全屏、搜索栏均适配小屏幕，悬浮球初始位置也会避开移动端顶栏
- **双语 i18n** — 中文/英文，自动检测

## 安装

**方法一** — 从酒馆界面安装（推荐）：

1. 打开 **扩展** → **安装扩展**
2. 输入仓库地址：`https://github.com/Asobi-123/memoir`
3. 刷新页面

**方法二** — 手动安装：

```bash
cd SillyTavern/data/default-user/extensions/
git clone https://github.com/Asobi-123/memoir.git
```

刷新 SillyTavern 页面即可使用。

---

## 使用方式

1. 点击右下角**悬浮球**打开面板（或在扩展设置页点「打开悬浮面板」）
2. **工作台**：填入楼层区间 → 生成总结 → 预览 → 确认采用
   这里的“楼层”就是聊天消息顺序，第一条是 1，第二条是 2。
3. **总结区**：查看状态、调整注入策略、编辑/融合已确认的总结正文
4. **设置**：选择主题、配置 API、选择/编辑总结预设、模板和成熟内容归档兼容补丁

### 提示词自定义

- 可以把你自己惯用的总结提示词模板直接放进来改。
- `成熟内容归档兼容补丁` 默认启用，也可以编辑、重置或关闭。
- `系统提示词` 没有必须保留的变量。
- `总结用户模板` 里请保留 `{{start_floor}}`、`{{end_floor}}`、`{{chat_history}}`，其余内容都可以改。

## 项目结构

```
manifest.json          — SillyTavern 扩展清单
index.js               — 入口
settings.html          — 扩展侧边栏入口
style.css              — 侧边栏样式
i18n/                  — 国际化翻译
src/
  i18n.js              — 自包含 i18n 模块
  utils.js             — 共享工具函数
  constants.js         — 常量
  bootstrap.js         — 启动
  core/                — 生命周期、上下文
  state/               — 设置、聊天数据
  services/            — LLM 路由、总结、注入、融合、哈希
  prompts/             — 预设、模板资产
  ui/                  — 浮动面板、Tab 页、表单绑定
docs/                  — 架构、数据模型、发布文档
```
